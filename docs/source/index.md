dartsort documentation ({{ release }})
======================================

`dartsort` is a spike sorter built around a new approach to probe drift and a statistical clustering framework. `dartsort` is also a modular Python toolkit for building spike sorting pipelines on top of [PyTorch](https://pytorch.org) and [SpikeInterface](https://spikeinterface.readthedocs.io/).
